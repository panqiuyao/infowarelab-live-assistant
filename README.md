# 直播助手

> 直播助手项目前端源码,vue-cli

## Build Setup

``` bash
#版本ｖ１
包含聊天室，用户页面

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
